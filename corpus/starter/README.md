# Starter Corpus

This directory is reserved for the first curated offline corpus used by Phase 1.

Initial rules:

- keep the starter corpus intentionally small
- prefer official, primary, or otherwise provenance-friendly documents
- store enough metadata alongside each document to satisfy the retrieval contract
- label user-supplied documents explicitly when they enter the corpus

Suggested future layout:

- `official/`
- `standards/`
- `legislation/`
- `user-supplied/`
